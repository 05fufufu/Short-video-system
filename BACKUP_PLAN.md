# 🆘 演示备用方案：局域网热点模式

当 cpolar 内网穿透不稳定或失效时，可以使用此方案。
此方案要求**演示用的电脑**和**访问网站的手机**连接同一个 Wi-Fi 热点（推荐使用手机开热点给电脑连）。

## 1. 获取电脑 IP 地址
1. 在电脑上打开 cmd。
2. 输入 `ipconfig` 并回车。
3. 找到 **无线局域网适配器 WLAN** (如果你是插网线就是以太网)。
4. 记录下 `IPv4 地址`，例如 `172.20.10.4`。

## 2. 修改配置文件
打开项目中的 `config/minio.go` 文件：

```go
// 找到这一行（cpolar 地址）并注释掉：
// MinioPublicServer = "6d2ffd2.r17.cpolar.top"

// 取消下面这行的注释，并填入你的 IP:8080
MinioPublicServer = "172.20.10.4:8080" 
```

> **注意**：IP 地址必须是你当前查到的那个，不要照抄 172.20.10.4，除非你的也是这个。

## 3. 重启服务
1. 在终端按 `Ctrl + C` 停止当前运行的后端。
2. 重新运行：`go run main.go`。

## 4. 手机访问
手机浏览器输入：`http://<你的电脑IP>:8080`
例如：`http://172.20.10.4:8080`

---

## 常见问题
*   **视频无法播放？** 
    *   确保 `config/minio.go` 改完后重启了服务。
    *   确保手机和电脑连的是同一个 Wi-Fi。
    *   检查电脑防火墙是否允许了 8080 端口（通常首次运行时会弹窗询问，选“允许”）。
*   **图片不显示？**
    *   同上，图片链接也是基于 `MinioPublicServer` 生成的。
